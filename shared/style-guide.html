HOW TO USE

Step 1: Link the Stylesheets

Add these two lines to the <head> of your reading HTML:

<link rel="stylesheet" href="../shared/base-styles.css">
<link rel="stylesheet" href="../shared/components.css">

Step 2: Remove Inline Styles

Delete the <style> block from your HTML (optional - only if you want to use the shared styles).

Step 3: Mix & Match Components

Use different class names from this guide to create variety:

  • Try .grid-3 instead of .grid-2
  • Use .card-floating for a softer look
  • Switch to .card-minimal for text-heavy readings
